## Usage: 

Point the "OffImage" and "OnImage" data property to your respective files. They should be put in the **Images** folder of your project. 


```
{
  "OffImage": "{PROJECT_FOLDER}offLed.png",
  "OnImage": "{PROJECT_FOLDER}onLed.png",
  "ShowMidiLabel": true
}

```
